## About

**FinlyAi** is the front-end of a finance management platform powered by AI. It enables users to track income and expenses, set budgets, visualize spending habits, and make smarter financial decisions — all from one intuitive dashboard.

---

## Features

- 💰 **Income & Expense Tracking** — Log and categorize every transaction effortlessly.
- 📊 **Visual Analytics** — Charts and graphs to understand your spending patterns at a glance.
- 🎯 **Budget Planning** — Set monthly budgets per category and get notified when you approach limits.
- 🤖 **AI-Powered Insights** — Personalized recommendations to optimize your finances.
- 🔒 **Secure & Private** — Your financial data is kept safe and confidential.
- 📱 **Responsive Design** — Works seamlessly on desktop, tablet, and mobile.

## Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/Sarvardev035/FinlyAi.git

# Navigate into the project directory
cd FinlyAi

# Install dependencies
npm install
```

### Running the App

```bash
# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

```bash
# Build for production
npm run build

# Preview the production build
npm run preview
```
